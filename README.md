# Smart Travel Alarm

**Smart Travel Alarm** is a Flutter-based mobile application that allows users to schedule alarms for specific times and locations â€” ensuring they never miss important travel reminders.  
It provides a modern UI, accurate time-zone scheduling, and instant/local notifications.

---

## Project Setup

Follow these steps to set up the project on your local environment.

### Clone the Repository
```bash
git clone https://github.com/tajemulcse/smart_travel_alarm.git
cd smart_travel_alarm 
```

## Technology Stack & Packages

Core Framework:

Flutter (using Dart)

Packages Used:

Location & Geo-Fencing: geolocator, geocoding

Notifications & Timing: flutter_local_notifications, timezone, intl

Data Persistence: shared_preferences

Media: video_player (for onbording videos)

System: permission_handler
